PLAstic

Closing the Loop on 3D Printing

PLAstic is a student-driven project that is focused on building a system that mechanically recycles PLA waste generated from 3D printers in the Corner Makery into functional filament that can be used for prototyping.

Who should join?

  • If you have interests in waste recovery and recycling
  • If you hate wasting money
  • If you want hands-on experience in electrical, mechanical, financial, outreach and design fields
  • If you want to help us make Northwestern more Sustainable

If you are interested, please email Samaher Nassar (SamaherNassar2028@u.northwestern.edu) for more info.
